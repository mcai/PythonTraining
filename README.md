# Python3 培训

## 文件层次说明

* doc/ - 培训文档。
* refs/ - 外部参考资料。
* data/ - MNIST手写体识别数据集，使用前请先将两个.tar.gz文件解压到当前路径下。
* *.m - MATLAB版本三层感知机进行MNIST手写体识别代码。
* *.py - Python3（numpy+matplotlib）版本三层感知机进行MNIST手写体识别代码。

## Python3代码使用方法

python3 ./tests_num_epochs.py

## MATLAB代码使用方法

使用MATLAB打开并运行tests_num_epochs.m即可。
